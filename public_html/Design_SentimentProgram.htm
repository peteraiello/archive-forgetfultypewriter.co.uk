- Sentiment Program

Negative Word Data

Through working and experimenting with the code, ideas developed by thinking about applying specific behaviour for groups of words, such as words of negative sentiment. After researching sentiment analysis, the interaction of removing negative words was designed using a negative word list of over 5000 words.

screenshot:sentiment example

By assigning certain behaviours to negative words, the program effectively removes words and leaves empty spaces in the text. Different types of data could be used for new interactions, such as removing adjectives or nouns.

Boolean isNegative(){

A boolean is used in a block of code which compares the strings of the words in the text with the list. The boolean can return true or false, and returns true if the word appears in the list.

if (isNegative() == true){

The result of the boolean can then be used to apply the different behaviours using an if statement; if the word is negative, then fade or fall way. A creative discovery made during experimenting with the code, is to reverse the result of the boolean to false, and effectively save words. This lead to further interactions such as removing one word per minute, with specific words saved.
