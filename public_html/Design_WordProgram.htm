- Word Program

screenshot example

The majority of the program's code is used to develop a simple and usable text editor. This consists of using the main program sketch with a word object class. ArrayLists feature heavily in the code, and are used in multiple ways, such as storing the letters, or strings for the words being typed. Certain methods define the behavior of the words, such as programming the words to fade or reappear, and are a creative element of the code.

void update (){
if(millis()> myRandom && ypos < height+50){
ypos = ypos +1;
}

The code used to animate the words, features inside the update method of the word class.

  • A random number is assigned to each word as it is typed (1004,2503, 9540 etc).
  • As the millis function, (a timer which returns the value of how long the program has been opened) increases, it causes the words to fall by using the above if statement:
  • Adding +1 to the ypos will make it move down, and this is how the individual words are animated in order to simulate falling text.

The update method can also be used to change the value of opacity (opacity=opacity+1;) to simulate fading text. Furthermore, millis values can be saved as new variables each time the mouse is pressed. This has culminated in using different flows of time for added functionality, such as starting and pausing the interaction during different stages.
