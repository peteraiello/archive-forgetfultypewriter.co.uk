This project is currently seeking participants from different backgrounds to help with testing the software.

user testing

- User Testing

Thus far, the software has been tested in different contexts, primarily with a student-based audience. This includes workshops with creative writing students, as well as individual sessions with the software in domestic settings. Sessions were initiated and lead by me. A manual/instruction guide was designed in order to walk participants through the interactions including information about the different forms of writing the software can be used for. A final manual of research was compiled, which features real examples of texts from user testing as well as quotes emphasising user experience (PDF available on request). Overall, users have cited the potential for the software to work in therapeutic and creative contexts. A high level of consideration was given to the sensitive nature of user engagement, which included therapeutic writing.

Areas of Use

  • Therapy The first area of interest involves the potential mental health benefits of using the software, such as whether use of the software can aid positive thinking.
  • Creative The second area of interest involves the creative purposes, and how engagement can illuminate curious new qualities of the text and how the process can add moments of contemplation.

Volunteer participants for testing these prototypes would be highly appreciated and further testing could help answer prevalent questions, including:

Participants

  • Routine Can the software be used effectively to help manage mood?
  • Integration Is it possible to integrate the software into the everyday practices of writing.
